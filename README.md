# Notion Fragments extension

GitHub: https://github.com/sanjarcode/notion-fragments

Demo (youtube): https://www.youtube.com/watch?v=IkGZR7V-XZg

Extensions store: (being reviewed by the Chrome store, use the workaround below)

# Workaround
- Download the code
- Extract the zip file
- Go to [chrome://extensions](chrome://extensions)
- Enable Developer mode
- Click "Load unpacked" and select the extracted folder
- Go to your Notion and it should be working. 
- Test: Clip https://github.com/donnemartin/system-design-primer using the Notion Clipper Extension, go to the clipped page, search for 'Start here', and click the link. it should scroll internally.
